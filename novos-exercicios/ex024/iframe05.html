Aprendendo a deixar iframe mais segura para captura de dados de sites de terceiros.

Usar sandbox"sandbox" para tirar todos os acessos do site de terceiro

Usar sandbox"allow-same-origin (para dizer que permite uso de conteudo da mesma origem) | allow-forms (para dizer que aceita captura de formularios) | allow-scrips (para dizer que aceira scripts)"

Usar referrerpolicy="no-referrer" (para não aceitar cookies de navegação ou sejá, não coletar nenhum dado do usuario do site.
