Aprendendo novos comandos de Input e parametros

input:date

"value" preenche o campo com a data indicada, sendo ano-mês-dia


input:month

"value" preenche o campo com o ano e mês indicada, sendo ano-mês


input:time

"value" preenche o campo com o horario indicada


Usando parametro "step" dentro do input:number

Na campo de Média, está começando de 0 e subindo de 0.1 decimos até 10. Isso tudo se dá devido ao uso do parametro step="0.1" dentro da tag ≶input:number>. Placeholder="0 a 10" / value="5" são outros parametros atribuidos para input. Caso sejá usado o parametro value="" ele substituirá o comando placeholder="", sendo assim o valor do campo de Média será automaticamente modificado para o comando de value="".
