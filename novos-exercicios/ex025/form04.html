Aprendendo novos parametros para form e input

Primeiro usamos dentro do form o parametro autocomplete="on" deixando o habilitado e dentro de input, usamos o parametro autocomplete="nickname" / "newpassword">

Usando o parametro required dentro do input para não deixar que o formulario sejá enviado em branco / sem preencher os campos de dados

Usando os parametros minlength="" e maxlength="" dentro do input para dizer o maximo e minimo de quantos caracters pode ter no campo de texto

Neste exemplo, iremos mostrar os conceitos anteriores e o parametro (size) dentro de input, para mostrar apenas no campo de texto com quantidade de um determinado numero de caracters na tela
